FUNDFLOW — DÉPLOIEMENT DOKPLOY SANS BUILD

1. Décompresse ce ZIP.
2. Dans un NOUVEAU dépôt GitHub, téléverse les fichiers extraits, pas le ZIP.
3. Dans Dokploy, crée un service DOCKER COMPOSE (pas Application).
4. Source : GitHub, branche main.
5. Compose Path : ./docker-compose.yml
6. Déploie. Aucun Dockerfile et aucun Nixpacks ne sont utilisés.
7. Ajoute le domaine sur le service fundflow, port 8000, puis redéploie.

Le code est intégré dans docker-compose.yml : aucun montage de fichier depuis le dépôt.
